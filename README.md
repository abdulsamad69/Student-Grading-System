Student Marksheet & Grading System (C Language)
A console-based application built to automate the process of marks entry, percentage calculation, GPA tracking, and grading for software engineering students. This project was developed during my 1st semester to understand data handling and conditional logic in C.
Features
Bulk Marks Entry: Input marks for 8 core subjects (Programming Fundamentals, Calculus, ICT, etc.).
Automated Calculations: Computes total marks and aggregate percentage instantly.
GPA & Grading Logic: Dynamically calculates GPA (up to 4.0) and assigns letter grades (A+ to F) based on university standards.
Tabular Transcript Output: Displays a clean, structured marksheet terminal view.
Concepts Used
Multi-dimensional Arrays (for storing course names)
One-dimensional Arrays (for storing student marks)
Loops (`for` loop for data collection and formatting)
String operations (`string.h` for handling grades)
Conditional Statements (`if-else` ladder for precise grading)
How to Run
Clone or download this repository.
Open `Marksheet.c` in any C compiler (e.g., Dev-C++, Code::Blocks, or VS Code).
Compile and Run the program.
